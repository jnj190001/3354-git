(c) What is the difference between git log and git status?

Answer: git log shows the commit logs and git status shows the working tree status.

(d) What is the difference between unstaged and untracked files?

Answer: Ustaged files are files that have not been marked for commit. Untracked files are files
that are not contained in the tracked repository.

(e) What is the difference between git checkout and git reset?

Answer: Git checkout is commonly used to discard changes in the working directory or switch betweeen 
brances while git reset is used to discard commits in a private branch, throw away uncomitted changes,
or to unstage a file.  

(f) What happens if you try to pull remote changes when there are uncommitted
changes in your working directory?

Answer: The uncommitted changes cannot be inculded in the pull. The changes will either need to be "stashed"
(i.e. saved for later) or the changes will need to be committed so that they will be committed in the pull.
